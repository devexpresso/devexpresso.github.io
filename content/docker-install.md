# Docker Install

### For Windows 10 Home

* Download **Docker Toolbox** from the following [link ](https://download.docker.com/win/stable/DockerToolbox.exe "Docker Toolbox for Windows")
* Install will add **Docker GUI**, **Oracle Virtual Box** and **Docker Quickstart**
